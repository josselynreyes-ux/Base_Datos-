Consultas, ¿dónde se guardan?
Las consultas no se guardan; solo se ejecutan cuando se las escribes.
Lo que sí se guarda es la información en las tablas de la base de datos.

Rendimiento con índices
Los índices hacen que las búsquedas sean más rápidas.
Es como poner marcadores en un libro para encontrar antes lo que necesitas.

Utilidad de las vistas
Las vistas muestran solo la información que te interesa sin tocar los datos originales.
Son como una “ventana” más ordenada y limpia para ver la información.

Importancia de las transacciones
Aseguran que un conjunto de cambios se haga completo o no se haga.
Evitan errores cuando ocurre un problema a mitad del proceso.

Uso de procedimientos
Sirven para guardar instrucciones que usas muchas veces.
Ahorran tiempo y evitan repetir el mismo código siempre.
